STL is Standard Template Library. <br>
STL has 4 parts - 
1. Containers
2. Iterators
3. Algorithms
4. Functors

## Containers 
Containers is a data structure that is pre implemented in STL. <br>
Containers has 3 types - 
- Sequential
    - vectors
    - stack
    - queue
    - pair (not a container)
- Unordered
    - unordered map
    - unordered set
- Ordered <strong>(store value in sorted order)</strong>
    - maps
    - multimap
    - set
    - multiset
